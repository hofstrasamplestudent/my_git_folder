CSS Grid Without Named Areas

This example demonstrates a grid layout using explicit grid lines instead of named areas.

Header
Sidebar
Main Content
Ad Section
Footer

HTML Code:

<div class="container">
    <div class="header">Header</div>
    <div class="sidebar">Sidebar</div>
    <div class="content">Main Content</div>
    <div class="ads">Ad Section</div>
    <div class="footer">Footer</div>
</div>

CSS Code:

.container {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: 80px auto 100px 60px;
    gap: 10px;
    max-width: 800px;
    margin: auto;
}

.header {
    grid-column: 1 / 3;
    grid-row: 1;
    background-color: lightblue;
}

.sidebar {
    grid-column: 1;
    grid-row: 2 / 4;
    background-color: lightgray;
}

.content {
    grid-column: 2;
    grid-row: 2;
    background-color: white;
}

.ads {
    grid-column: 2;
    grid-row: 3;
    background-color: lightcoral;
}

.footer {
    grid-column: 1 / 3;
    grid-row: 4;
    background-color: lightgreen;
}

@media (max-width: 600px) {
    .container {
        grid-template-columns: 1fr;
        grid-template-rows: 80px auto auto auto 60px;
    }
    .header {
        grid-column: 1;
        grid-row: 1;
    }
    .sidebar {
        grid-column: 1;
        grid-row: 2;
    }
    .content {
        grid-column: 1;
        grid-row: 3;
    }
    .ads {
        grid-column: 1;
        grid-row: 4;
    }
    .footer {
        grid-column: 1;
        grid-row: 5;
    }
}
