CSS Grid with Named Areas

Header
Sidebar
Main Content
Ad Section
Footer

HTML Code:

<div class="container">
    <header class="header">Header</header>
    <nav class="sidebar">Sidebar</nav>
    <main class="content">Main Content</main>
    <aside class="ads">Ad Section</aside>
    <footer class="footer">Footer</footer>
</div>

CSS Code:

.container {
    display: grid;
    grid-template-areas: 
        "header  header"
        "sidebar content"
        "sidebar ads"
        "footer  footer";
    grid-template-columns: 200px 1fr;
    grid-template-rows: 80px auto 100px 60px;
    gap: 10px;
}

.header { grid-area: header; background-color: lightblue; }
.sidebar { grid-area: sidebar; background-color: lightgray; }
.content { grid-area: content; background-color: white; }
.ads { grid-area: ads; background-color: lightcoral; }
.footer { grid-area: footer; background-color: lightgreen; }
