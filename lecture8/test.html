Semantic HTML with Named Grid Areas

This live example uses semantic HTML elements combined with CSS Grid named areas to create a flexible layout. Below you can see both the live website and its HTML/CSS code.

Website Header

  • Home
  • About
  • Contact

Main Content

This is the main content area. Notice how the semantic <main> element is assigned to the "main" grid area.

Sidebar

This sidebar provides additional information and is placed using its named grid area.

Website Footer

HTML Code:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Semantic HTML with Named Grid Areas</title>
  <style>
    /* CSS code here */
  </style>
</head>
<body>
  <div class="container">
    <header>
      <h1>Website Header</h1>
    </header>
    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
      </ul>
    </nav>
    <main>
      <h2>Main Content</h2>
      <p>This is the main content area.</p>
    </main>
    <aside>
      <h2>Sidebar</h2>
      <p>Additional info here.</p>
    </aside>
    <footer>
      <p>Website Footer</p>
    </footer>
  </div>
</body>
</html>

CSS Code:

body {
  font-family: sans-serif;
  margin: 20px;
  background: #f9f9f9;
  color: #333;
  text-align: center;
}
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "nav main"
    "nav aside"
    "footer footer";
  grid-template-columns: 200px 1fr;
  gap: 10px;
  max-width: 800px;
  margin: auto;
  border: 2px solid #ccc;
  padding: 10px;
}
header {
  grid-area: header;
  background: lightblue;
  padding: 10px;
}
nav {
  grid-area: nav;
  background: lightgray;
  padding: 10px;
}
main {
  grid-area: main;
  background: white;
  padding: 10px;
}
aside {
  grid-area: aside;
  background: lightcoral;
  padding: 10px;
}
footer {
  grid-area: footer;
  background: lightgreen;
  padding: 10px;
}
